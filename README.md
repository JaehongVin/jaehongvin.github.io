# 흔한 개발자의 발자취

개인 포트폴리오 모노레포. 블로그를 포함한 여러 앱과 공유 패키지를 관리합니다.

> https://jaehongvin.github.io

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | Next.js 16.1 (Static Export) |
| 언어 | TypeScript 5.9 |
| 런타임 | Node.js 24, React 19 |
| 스타일링 | Tailwind CSS 4 |
| 콘텐츠 | MDX + next-mdx-remote |
| UI 컴포넌트 | Radix UI + CVA |
| 모노레포 | pnpm 10 Workspaces + Turborepo |
| 코드 품질 | Biome + Husky |
| 배포 | GitHub Actions → GitHub Pages |
