## 소개

안녕하세요, 프론트엔드 개발자 **빈재홍**입니다.

이 블로그는 개발하면서 배운 것들을 기록하고 공유하기 위해 만들었습니다. 주로 프론트엔드 기술에 대한 글을 작성할 예정이지만, 가끔은 일상적인 이야기도 나눌 수 있으면 좋겠습니다.

## 블로그를 만든 이유

개발자로서 성장하기 위해 가장 중요한 것 중 하나는 **배운 것을 정리하고 기록하는 것**이라고 생각합니다.

### 기록의 중요성

- 나중에 다시 찾아볼 수 있는 **레퍼런스**가 됩니다
- 글로 정리하면서 **이해도가 깊어집니다**
- 다른 개발자들과 **지식을 공유**할 수 있습니다

### 왜 직접 만들었나?

Velog, Medium 같은 플랫폼도 좋지만, 직접 블로그를 만들면서 얻는 것들이 있습니다:

1. 최신 기술 스택을 직접 사용해볼 수 있습니다
2. 원하는 대로 커스터마이징할 수 있습니다
3. 포트폴리오로도 활용할 수 있습니다

## 기술 스택

이 블로그는 다음 기술들로 만들어졌습니다:

- **Next.js 16** - React 기반 풀스택 프레임워크
- **React 19** - 최신 React 버전
- **TypeScript** - 타입 안정성
- **Tailwind CSS v4** - 유틸리티 기반 스타일링
- **MDX** - 마크다운 + JSX

앞으로 이 블로그에서 다룰 주제들입니다:
