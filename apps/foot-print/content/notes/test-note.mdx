## 테스트 노트입니다

이 노트는 노트 기능이 정상적으로 동작하는지 확인하기 위한 테스트 게시글입니다.

---

## 기능 확인 항목

- 노트 리스트 페이지에서 이 글이 보이는지 확인
- 노트 상세 페이지에서 MDX 렌더링이 정상인지 확인
- 카테고리, 태그 필터링이 동작하는지 확인
- 목차(TOC)가 정상적으로 생성되는지 확인

### 코드 블록 테스트

```typescript
const greeting = "Hello, Notes!";
```

### 인라인 코드 테스트

`inline code`가 정상적으로 렌더링되는지 확인합니다.
